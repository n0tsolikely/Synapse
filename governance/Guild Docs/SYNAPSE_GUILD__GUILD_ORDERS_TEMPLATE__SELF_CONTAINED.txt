================================================================================
0) DEFINITIONS (CANONICAL SHORT FORM)
================================================================================

- Guild Orders:
  - Authoritative statement of WHAT is being built now and DONE criteria.
- Raid:
  - Active execution run of one Guild Orders artifact.
  - State by location: ACTIVE / PAUSED / COMPLETED.
- Dungeon:
  - Major scope slice within Guild Orders.
  - Decomposes into one or more Quests.
- Quest:
  - Atomic executable unit with deterministic verification.

6) EXECUTION POLICY
================================================================================

- R0/R1: execute automatically in EXECUTE mode.
- R2+: explicit consent once per batch.
- No silent scope expansion.
- If contradiction found, pause and resolve via Control Sync.
