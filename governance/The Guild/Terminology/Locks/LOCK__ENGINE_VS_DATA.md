# LOCK — Engine vs Data
**Locked:** 2026-01-31 19:20:21 (America/Montreal assumed)

## Canon meaning
- Engine = portable, stateless canon package dropped into any AI context.
  It must not contain session-specific snapshots/audits/private history.

- Data = stateful continuity for a specific Subject:
  snapshots, audits, discoveries, codex drafts, notes, etc.

## Purpose
Prevents tainting the Engine and enables clean reuse across AIs and sessions.

## Do not assume
- Engine is not where evolving session history lives
- Data is not where canon rules are rewritten (unless explicitly versioned and promoted)
