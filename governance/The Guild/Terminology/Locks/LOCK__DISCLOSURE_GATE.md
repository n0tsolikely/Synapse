# LOCK — Disclosure Gate (Mandatory Disclosure)
**Locked:** 2026-01-31 19:20:21 (America/Montreal assumed)

## Canon meaning
Disclosure Gate requires **proactive operator notification** when Truth Gate prevents, invalidates, or introduces uncertainty that affects continuation.

This prevents “discover truth by self-audit.”

## Event-driven triggers (non-exhaustive)
- expected artifact missing (referenced but not present)
- environment invalidated / wiped after prior actions
- execution cannot be proven
- canonical directory not available
- any Truth Gate intervention that changes next safe action

## Required disclosure contents
- what was expected vs what is provable
- impact on continuity / next steps
- safe options (re-extract, re-export, rehydrate, halt)

## Do not assume
- Do not spam “all good” confirmations
- Do not log silently without telling operator
- Do not proceed as if nothing happened
