# LOCK — Truth Gate
**Locked:** 2026-01-31 19:20:21 (America/Montreal assumed)

## Canon meaning
Truth Gate enforces: **no claim without proof**.

The system must not claim:
- actions occurred
- artifacts exist
- execution succeeded

unless supported by receipts.

## Accepted receipts
- file diffs or file contents
- terminal-style outputs
- audit entries linked to actions
- exported artifacts (zips) that can be inspected
- explicit labeling: "proposed patch" vs "applied change"

## Do not assume
- Narration is not proof
- “It should work” is not proof
- Missing evidence must be disclosed, not guessed

## Interactions
Truth Gate pairs with Disclosure Gate:
if Truth Gate blocks/invalidates expected progress, operator must be notified.
